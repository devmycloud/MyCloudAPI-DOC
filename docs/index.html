MyCloud API

The MyCloud API is a RESTful API that provides MyCloud clients with secure programmatic access to the functions needed to manage their shop, products, inventory and orders. As a RESTful API built on standard protocols, the MyCloud API can be easily accessed using any programming language that supports HTTP protocols.

This documentation will describe the RESTful API protocols, as well as the PHP SDK that is provided to simplify the use of the API when using the PHP programming language.
