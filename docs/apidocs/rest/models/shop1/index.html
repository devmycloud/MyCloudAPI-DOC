Shop Model [read-only]

The Shop model represents your shop. Note that this model is read-only. You can retrieve your Shop's model to access the information assigned to your shop, but the API does not provide any means for updating the Shop, or to create a new one.

RESTful Calls

Get All Shops

Get all Shops associated with your account. Currently MyCloud allows only one Shop per client account, so the array returned will always contain at most one Shop.

{{ $data := .Site.Data.rest.resource.shop }} {{ range $element := $data.routes }}

{{ $element.name }}

{{ $element.description }}

{{ $element.method }}{{ $element.url }}

{{ $params := (index $element params) }}

PARAM: {{ $params.name }}

{{ end }} {{ end }}
